# PSY6422_Project

### What this project does:

This project creates a visualisation of publicly available neuroimaging data by plotting the frequency of different modalities such as EEG, fMRI and NIRS over time from 2018 to 2025.

### Version Control

This project uses R version 4.5.1 (2025-06-13). All coding was conducted on macOS Sequoia 15.3.2. For more information about the packages and package versions used in this project please refer to section 1.2 of the PSY6422_Project webpage linked in the about section of this repo.
